Waddupboard

Front end Web Development
April 2018

Background

Waddupboard is a web application that allows users to attach notes and organize them however they want on a board. Again, the front end was built on React and the backend with Firebase. This was another assignment for my class CS52.

My Contribution

Since this was an assignment for class, I worked on this alone. I designed the front end and created data models that would be used in Firebase. The database kept track of the location, text/images, and color of each card.

What Went Well

I had prior knowledge using Firebase. It was really easy to get started with the API since the tool has such great documentation.

What Could Have Gone Better

I had an issue with live updating the position of each card. I struggled with coming up with a solution that would update the position of each card in realtime without have to make dozens of calls to Firebase.

What I Learned

Sometimes brute force is the answer. This doesn't apply to every single tasks assigned; however, in the scope of assignment sometimes getting something to work is better than having nothing at all.
